# Website bán túi xách
## Giới thiệu
Website bán túi xách được xây dựng bằng ASP.NET Web Forms và SQL Server.
## Chức năng
* Đăng ký / đăng nhập
* Hiển thị sản phẩm
* Tìm kiếm sản phẩm
* Giỏ hàng
* Đặt hàng
* Quản lý đơn hàng
## Công nghệ sử dụng
* C#
* ASP.NET Web Forms
* SQL Server
* HTML/CSS
* JavaScript
## Hướng dẫn chạy
1. Restore database từ file .bak
2. Mở file .sln bằng Visual Studio
3. Chạy bằng IIS Express
