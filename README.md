# Website bán túi xách
## Giới thiệu
Đây là website bán túi xách được xây dựng bằng ASP.NET Web Forms và SQL Server nhằm hỗ trợ người dùng mua sắm trực tuyến và quản lý sản phẩm.
## Chức năng chính
### Người dùng
* Đăng ký / đăng nhập tài khoản
* Xem danh sách sản phẩm
* Tìm kiếm sản phẩm
* Xem chi tiết sản phẩm
* Thêm sản phẩm vào giỏ hàng
* Đặt hàng và thanh toán
### Quản trị
* Quản lý sản phẩm
* Thêm / sửa / xóa sản phẩm
* Quản lý đơn hàng
* Quản lý khách hàng
## Công nghệ sử dụng
* C#
* ASP.NET Web Forms
* SQL Server
* HTML/CSS
* JavaScript
* SweetAlert
* GitHub
## Hướng dẫn chạy project
1. Clone repository:
```bash
git clone https://github.com/nghiathang6/Webbantuisach.git
```
2. Restore database từ file `.bak`
3. Mở file `.sln` bằng Visual Studio
4. Cấu hình connection string trong `Web.config`
5. Chạy project bằng IIS Express
